
🔎 프로젝트 개요

인증 기반 키 관리 시스템을 개발했습니다.

JWT 인증 구조를 기반으로 키 생성, 관리, 갱신 기능을 구현했으며
해당 시스템은 신속확인제도 및 조달청 심사를 통과하여
공공 서비스로 등록되었습니다.

🛠 기술 스택

Java

Spring Boot

Spring Security

JWT

React

MariaDB

✨ 주요 기능

JWT 기반 사용자 인증

키 생성 / 재발급 / 삭제

키 만료 및 갱신 관리

사용자 권한 기반 접근 제어

🧠 구현 포인트

보안 기준을 고려한 인증 구조 설계

예외 상황(만료, 위조 토큰) 처리

실제 운영 환경을 고려한 안정성 확보

프론트엔드와 API 연동을 통한 관리 화면 제공

🧑‍💻 담당 역할

백엔드 API 설계 및 구현

JWT 인증 로직 개발

프론트엔드 키 관리 화면 구현

보안 및 심사 기준 대응
